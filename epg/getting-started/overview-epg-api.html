introduction

The EPG API allows you to retrieve an electronic programming guide for a Cloud Playout channel. Cloud Playout channels are created in Studio - see Using Cloud Playout to Stream VOD Videos for details.

Authentication

The EPG API uses the Brightcove OAuth system to authenticate requests through an access token that is passed in an authorization header with the request:

Authorization: Bearer {access token}

Access tokens are retrieved using the OAuth API - see Getting Access Tokens for details. You will also need client credentials to authenticate the requests for access tokens. API URL

The URL for the EPG API request is:

https://sm.cloudplayout.brightcove.com/accounts/{account_id}/channels/{channel_id}/epg
