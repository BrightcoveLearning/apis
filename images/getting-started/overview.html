Introduction

The Brightcove Image API provides a quick and easy way to resize, crop, and/or rotate an image using a simple GET request.

Request URL

The request URL for the Image API is:

https://images.brightcovecdn.com/image/v1/account_id/default/url/{image_url}

The required path parameters are:

  • account_id: your Video Cloud account id
  • image_url: a public URL where the image can be found
Note that the image_url must be properly encoded. If you use JavaScript, use the encodeURIComponent() method. You can also use the String Encoder provided by Brightcove Learning Services:

URL parameters

To get a transformed version of your image, use one or more of the following URL parameters with your request:

URL Params for Image API
Parameter Description
resize Resize the image by setting this equal to "widthxheight" - example: resize=160x90 (can be combined with rotate; if combined with crop, only the resizing is performed)
crop Crop the width and height by setting this equal to "widthxheight" - example: crop=160x90 - the origin point is the center of the image. (Can be combined with rotate; if combined with resize, only the resizing is performed.)
rotate rotate the image clockwise by the specified number of degrees - example: rotate=90 - positive or negative numbers can be used. (Can be combined with resize or crop; if combined with both, only the resizing and rotation is performed.)
